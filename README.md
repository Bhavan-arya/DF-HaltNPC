### Features
- **Morph:** Seamless transition from Globe to Plane via scroll.
- **Interact:** Hover to emit light, drag to rotate perspective.
- **Physics:** Organic simplex noise and reactive particle systems.
- **Aesthetics:** Glassmorphism UI meets raw data visualization.
